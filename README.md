BLASTIUM
aka BLASTACULAR
aka BRASS TACKS

Summary:
A small topdown space shooter inspired by Asteroids.

Controls:
W - accelerate
S - brake/decelerate
A - turn left/counter-clockwise
D - turn right/clockwise
; - fire blaster
' - fire moonshot
P - pause game
M - mute
