Blastium
aka Blastacular
aka Brass Tacks

Summary:
A small topdown space shooter inspired by Asteroids.

Controls:
  Ship Movement:
    W - accelerate
    S - brake/decelerate
    A - turn left/counter-clockwise
    D - turn right/clockwise

  Weapons:
    ; - fire blaster
    ' - fire moonshot

  Misc:
    P - pause game
    M - mute

Credits:
  Developer - Daniel Purdes @danielpuedes
  Fonts - Chris Early @chriswearly

Engine:
  Love2d - https://love2d.org/
