Blastium
aka Blastacular
aka Brass Tacks

Summary:
A topdown space shooter inspired by Asteroids & Dark Souls.

Controls:
  Ship Movement:
    W - accelerate
    S - brake/decelerate
    A - turn left/counter-clockwise
    D - turn right/clockwise

  Weapons:
    ; - fire blaster
    ' - fire moonshot

  Misc:
    P - pause game
    M - mute
